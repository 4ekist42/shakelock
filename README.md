# ShakeLock

Минимальное Android-приложение, которое блокирует устройство при тряске. Порог срабатывания настраивается.

## Возможности
- Блокировка при тряске с настраиваемым порогом
- Блокировка через **Device Admin** (`DevicePolicyManager.lockNow()`)
- Работа в фоне через **foreground service**
- **Не блокирует при выключенном экране** (на локскрине блокирует, т.к. экран включён) 

## Сборка (debug APK)
Требования:
- Android SDK + platform-tools (adb)
- В проекте есть Gradle Wrapper

Собрать APK:
```bash
./gradlew :app:assembleDebug
```

Где лежит APK:
```text
app/build/outputs/apk/debug/app-debug.apk
```

Установка на устройство через ADB:
```bash
adb install -r app/build/outputs/apk/debug/app-debug.apk
```

## Использование
1. Открой приложение и выставь порог тряски.
2. Нажми **Start**.
3. Включи **Администратора устройства (Device Admin)** в системном окне (нужно для блокировки).
4. Нажми **Start** ещё раз.
5. Потряси телефон при включённом экране → устройство заблокируется.

## Примечания
- На Android 13+ может запросить разрешение на уведомления (для уведомления foreground service).
- Порог считается по линейному ускорению (гравитация отфильтрована), поэтому в покое значение близко к 0.
